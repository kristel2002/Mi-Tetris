📝 Instalación y ejecución:

# Clonar el repositorio
git clone https://github.com/kristel2002/Mi-Tetris.git

# Entrar al directorio
cd Mi-Tetris

# Instalar dependencias
pip install pygame

# Ejecutar el juego
python proyecto_2.py

🎥 ¿Cómo se juega?

1. Las piezas caen desde la parte superior
2. Usa las flechas para mover y rotar
3. Completa líneas horizontales para ganar puntos
4. ¡El juego termina cuando las piezas llegan a la cima!

🐍 Desarrollado con Python 3.11 y Pygame
